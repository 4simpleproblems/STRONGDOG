Vignette Ad Vote

This is a vote on whether or not StrongDog should add vignette ads. A vignette ad is an ad that appears in between page views (for example, switching to a different game). It’s like a popup ad, but you can close it immediately and only see it about once an hour. Adding these ads could potentially double StrongDog’s revenue, which would help us:

  • Host a better proxy for apps like Discord.
  • Add more games (e.g. "Human or Not").
  • Buy more domain names.
